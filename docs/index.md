# Ueberblick Dokumentation

**Ueberblick** ist eine Anwendung zur geografischen Datenerfassung mit Offline-Unterstuetzung. Die Admin-Oberflaeche heisst **Ueberblick Sector**, die Teilnehmer-App heisst **Ueberblick**.
